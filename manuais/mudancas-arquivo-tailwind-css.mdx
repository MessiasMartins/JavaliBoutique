Durante o desenvolvimento, foi necessária a criação de classes personalizadas, pois as classes padrão
do tailwind não continham as configurações necessárias.

Foram adicionadas as seguintes classes ao arquivo tailwind.css:

    @layer utilities {
    .ml-45 {
        margin-left: 45rem
  }
}

    @layer utilities {
    .text-62 {
        font-size: 62px
    }
    }

    @layer utilities {
    .text-48 {
        font-size: 48px
    }
    }

    
    Essa classe faz o carrossel rolar mais devagar
    @layer utilities {
    .roll-5 {
        transition: all ease 0.5s
  }
}
